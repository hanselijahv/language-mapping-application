Welcome to Our Interactive Philippines Language Map for Luzon

Our web app shows the language mapping of Luzon, visually representing the distribution of languages spoken across the largest island in the Philippines. It uses interactive maps to display linguistic data, allowing users to explore which languages are spoken in specific provinces, often highlighting dominant languages and linguistic diversity. This includes languages like Tagalog, Ilocano, Bikol, Kapampangan, Pangasinan, and others, reflecting the island's rich linguistic landscape.

How to Use:

  1. Access the Web App: Navigate to Luzon. Zoom in or use the search bar to find specific provinces like "Metropolitan Manila" or "Benguet".
  2. Explore Language Data: Click on a province to view its description and dominant languages. A pie chart will display language percentages.
  3. Analyze Linguistic Diversity: Compare diversity across provinces (e.g., multilingual Ifugao vs. monolingual Bulacan).
  4. Interact with Search Feature: Use the search bar or scroll the preset province list.
