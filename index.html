Welcome to Our Interactive Philippines Language Map for Luzon

Our web app that shows the language mapping of the whole Luzon, visually representing the distribution of languages spoken across Luzon, the largest island in the Philippines. It uses interactive maps to display linguistic data, allowing users to explore which languages are spoken in specific provinces, often highlighting dominant languages and linguistic diversity. For Luzon, this would include mapping languages like Tagalog, Ilocano, Bikol, Kapampangan, Pangasinan, and others, reflecting the island's rich linguistic landscape.

How to Use:

  1. Access the Web App: Navigate to Luzon: Once the map loads, zoom into Luzon, which is the northernmost major island group of the Philippines. You can do this by scrolling or using the search bar to focus on specific provinces like "Metropolitan Manila" or "Benguet".
  2. Explore Language Data: Click on a province to view details about the description and languages spoken there. For example, clicking on Albay in the Bicol Region will show the description of Albay, and a few scrolls down that Bicolano (Central Bikol) is the dominant language, with Tagalog also widely spoken. Beside the map, a pie chart shows the percentages for each language.
  3. Analyze Linguistic Diversity: Use the mapâ€™s features to compare linguistic diversity across provinces. For instance, Ifugao in the Cordillera region is highly multilingual, with over half the households speaking Tuwali or Ayangan, while provinces near Metro Manila, like Bulacan, are more monolingual, with Tagalog as the primary language.
  4. Interact with Search Feature: If you are not familiar with the location of the provinces, you can use the search bar and type the province you want to locate or you can scroll down the preset provinces that are arranged alphabetically.
